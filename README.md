# Homebrew TRS-80 Restoration

This archive contains supporting files for our ongoing restoration of the Homebrew Mystery TRS-80 machine, as featured in my YouTube video series: [![YouTube Channel Subscribers](https://img.shields.io/youtube/channel/subscribers/UCPIggfu0vP9aWZfYNkmdsYA)](https://www.youtube.com/channel/UCPIggfu0vP9aWZfYNkmdsYA)

[![Homebrew Mystery TRS-80 YouTube playlist](https://img.youtube.com/vi/_C0M3T0IJHc/0.jpg)]([https://www.youtube.com/watch?v=465Awa2zUTM](https://www.youtube.com/playlist?list=PLIfOQebaUAyHaB6TcVJD4BRhi-J4jPYv-))


## Repository Contents

You will find the following here:

### ROMs/

The dumped ROMs from the board labeled "CPU".

### ROMs/ref/

Reference files to help us compare the ROMs to known ROMs from other TRS-80 contemporary computers to help us identify what this mystery machine actually us.

### img/

High resolution images of the Homebrew TRS-80 that some of you have asked for :-)
